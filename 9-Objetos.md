# Objetos

Objetos são tipos de dados que guardam informações denominadas atributos. A sua sintaxe está a seguir:

```
var objeto = {
    atributo: valor,
    atributo: valor,

    ...

};
```

Podemos acessar os atributos de um objeto da seguinte maneira:
> objeto.atributo

ou
>objeto["atributo"];
